# Currency Converter
Welcome to Currency Conveter, a streamlined and intuitive website dedicated to simplifying currency conversions. This project was developed as a training task for MAD Solutions, showcasing a modern web application built with the latest technologies.

## Features
- Real-Time Currency Conversion: The site allows users to convert between a wide array of global currencies in real-time, providing up-to-date exchange rates for accurate calculations from a wildly known API with high availability.
- Responsive Design: Whether you're on a desktop, tablet, or smartphone, ExchangePro delivers a consistent and engaging experience, adapting flawlessly to any screen size.
- Historical Rate Charts: Dive deeper into currency trends with access to historical exchange rate data, enabling users to make informed decisions based on past performance. (**Work in progress**)

## Technologies
- React app built with Vite
- Vanilla CSS
- [Exchange Rate API](https://www.exchangerate-api.com)
